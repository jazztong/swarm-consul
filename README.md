# Swarm consul

Setup Consul in Docker Swarm

## Install Vagrant

[Install here](https://www.vagrantup.com/downloads.html)

## Setup Vagrant

```
vagrant up
```

## Login Vagrant

```
vagrant ssh <name>
```

## Kill Vagrant

```
vagrant destroy -f
```

## Default Vagrant

Vagrant box will setup 2 managers and 2 workers and join all as Swarm mode
